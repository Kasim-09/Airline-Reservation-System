# Airline Reservation System (Java)

A **console-based Airline Reservation System** built in **Java** using **JDBC and MySQL (XAMPP)**.  
This project allows users to manage **flights, passengers, and ticket reservations** with persistent database storage.

## Features
- Add, view, update flights
- Book flight tickets
- Cancel reservations
- View passenger details
- Display all bookings
- Search flights
- Check seat availability
- Data stored in MySQL database

## Tech Stack
- Java
- JDBC
- MySQL (XAMPP)
- Object-Oriented Programming (OOP)
- SQL
- Scanner for input

## Requirements
- Java JDK 8+
- XAMPP (MySQL running)
- MySQL Connector/J (JDBC driver)

## Database Setup
1. Start **XAMPP** and run **MySQL**
2. Create database:
   ```sql
   CREATE DATABASE airline_db;
